Rich - The easy to use rich text editor built on content editable

What is content editable? Content editable is an attribute that lets you manipulate the text on elements like you would a text area. With the execCommand function you are opened to a wide range of native support for rich text editing.

Rich is built with extension in mind, you can create your own icons that will execute specific commands in document.execCommand. If you find any bugs or have any issues please report them on github.
